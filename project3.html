Project 3

Photo Alphabet

Adobe Photoshop

Project Description: I first chose a source image as inspiration to create the letters of the alphabet. I chose colorful popsicle sticks. Using Adobe Photoshop, I used many different tools to create letters of the alphabet using the source image of the popsicle sticks. I morphed the popsicle sticks to curve and make different shapes to follow the way the letters were formed. I used different colored popsicle sticks together to make it a little more fun and kid-friendly.
